# 数据×日报交叉分析框架

## 分析流程

```
Step 1: 采集四平台后台数据
         ↓
Step 2: 读取飞书行业日报
         ↓
Step 3: 提取关键行业信号（3-5个）
         ↓
Step 4: 后台数据 ↔ 行业信号交叉验证
         ↓
Step 5: 生成分平台洞察+优化建议
         ↓
Step 6: 输出跨平台行动清单（P0/P1/P2）
         ↓
Step 7: 生成HTML复盘报告
```

## 行业信号分类

### 信号类型1：市场趋势变化
**示例：** 防晒市场274.5亿同比降7.8%，存量博弈
**数据验证：** 公众号阅读量是否下降？小红书曝光是否波动？
**优化方向：** 内容差异化、技术叙事、高端定位

### 信号类型2：技术/产品创新
**示例：** AI+防晒成为2026夏季关键词，戴可思推出6月龄+物理防晒
**数据验证：** 技术科普内容表现如何？（如「独家定制原料」仅21阅读）
**优化方向：** 升级技术叙事、蹭热点、实测对比

### 信号类型3：政策/标准变化
**示例：** 青少年护肤无强制性国标，标准制定窗口期
**数据验证：** 青少年相关内容数据（今早直播1,326观看/¥2,133成交）
**优化方向：** 权威内容建立心智、参与标准制定宣传

### 信号类型4：竞品动态
**示例：** 红色小象综艺冠名、一页科研背书增长117%
**数据验证：** 自家内容策略是否需要调整？
**优化方向：** 借鉴竞品成功模式、差异化竞争

### 信号类型5：营销节点
**示例：** 618提前打响，天猫新品增长33%
**数据验证：** UGC活动数据（晒娃有奖互动率5.1%）
**优化方向：** 启动UGC活动、618预热内容、直播排期

### 信号类型6：明星/KOL动态
**示例：** 田栩宁代言玫珂菲、黄宇航抖音合作
**数据验证：** 自家是否有代言人相关数据？
**优化方向：** 借势热点、关联品牌定位、转发互动

## 交叉分析模板

### 信号卡片格式

```markdown
**🔴 信号X：[信号标题]**
- **行业数据**：[具体数字/趋势]
- **后台验证**：[对应平台数据]
- **差距/机会**：[我们做得如何？]
- **→ 优化方向**：[具体行动]
```

### 示例

```markdown
**🔴 信号1：防晒市场进入"存量博弈+技术突围"阶段**
- 行业：2025年防晒零售274.5亿同比降7.8%，TOP5集中度36.51%
- 后台：公众号「原料科普」21阅读 vs 「避坑指南」637阅读
- 差距：技术科普内容有需求，但标题不够"痛点化"
- → 优化方向：升级「AI防晒黑科技」技术叙事，标题改为「为什么海龟爸爸小光盾能全波段防护？」
```

## 分平台分析维度

### 公众号
| 维度 | 分析要点 |
|------|---------|
| 内容效率 | 阅读/分享/收藏比例，什么内容留存率高 |
| 社交裂变 | 朋友圈 vs 聊天分享占比，裂变系数 |
| 算法推荐 | 推荐流量占比，标题关键词优化空间 |
| 直播转化 | 观看人次/成交金额，私域变现效率 |
| 行业对标 | 竞品公众号内容策略，差异化空间 |

### 小红书
| 维度 | 分析要点 |
|------|---------|
| 内容转化 | 曝光→观看→互动→涨粉，漏斗效率 |
| UGC效率 | 激励型内容 vs 纯内容型，互动率对比 |
| 封面优化 | 点击率差异，视觉元素分析 |
| 粉丝质量 | 来源分布，搜索/推荐/主页比例 |
| 行业对标 | 竞品KOL合作策略，母婴垂类 vs 萌娃类 |

### 视频号
| 维度 | 分析要点 |
|------|---------|
| 内容产出 | 作品数量/频率，是否满足平台算法偏好 |
| 互动率 | 播放/点赞/评论比例，内容吸引力 |
| 直播回放 | 精华片段二次利用效率 |
| 发布时间 | 黄金时段17:00-19:00利用率 |
| 行业对标 | 短视频种草趋势，直播矩阵 |

### 微博
| 维度 | 分析要点 |
|------|---------|
| 产出稳定性 | 日更达成率，脉冲式 vs 稳定式 |
| 粉丝精准度 | 人均阅读量，高质量粉丝占比 |
| 蓝V权重 | 认证状态，年审及时性 |
| 热点借势 | 明星/KOL关联，转发抽奖效率 |
| 行业对标 | 竞品微博策略，红色小象15.5万粉玩法 |

## 行动清单优先级

### P0 · 立即执行（48小时内）
- 直接影响收入/流量的紧急事项
- 行业热点窗口期（如618预热）
- 账号风险（如蓝V到期）

### P1 · 本周完成
- 内容策略调整
- 优化建议落地
- UGC活动启动

### P2 · 本月规划
- 长期内容规划
- 跨平台联动
- 竞品对标学习

## 复盘报告结构

### 必须包含的模块

1. **核心发现**（3个行业信号 × 数据交叉验证）
2. **四平台速览**（昨日核心指标卡片）
3. **分平台详细复盘**（数据表格 + 洞察框 + 优化建议）
4. **竞品对标**（仅小红书账号策略，去掉电商销售数据）
5. **跨平台策略**（P0/P1行动清单）
6. **监控指标**（本周目标量化）

### 颜色规范
- 🟢 绿色 insight-box：积极发现、增长机会
- 🟡 黄色 warning-box：需要关注、优化建议
- 🔴 红色 danger-box：紧急风险、立即行动
- 🔵 蓝色 info-box：行业信息、参考数据

## 输出文件命名规范

- 数据看板：`media_dashboard_full_YYYY-MM-DD.html`
- 复盘报告：`daily_review_YYYY-MM-DD.html`
- GitHub首页：`index.html`（覆盖更新）
