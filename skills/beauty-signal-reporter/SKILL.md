# Beauty Signal 小红书数据周报

## 快速使用

### 生成周报
```
/周报 [周次] [数据截图]
```
- 璐婷发送截图后，提取数据填入模板
- 生成 HTML → 导出 PDF
- 发布到本群

### 查看采集指南
```
/采集指南
```
- 输出「每周需要提供哪些截图」的完整清单
- 截图规范和时间要求

### 查看示例周报
```
/示例周报
```
- 输出一份填充了模拟数据的完整周报
- 供璐婷确认格式和内容是否符合预期

## 文件结构

```
beauty-signal-reporter/
├── SKILL.md                    # 本文件
├── 数据采集指南.md              # 璐婷的截图采集规范
├── 周报模板.html               # HTML模板（Chart.js可视化）
├── 示例周报.html               # 填充模拟数据的示例
└── references/
    └── 美妆行业benchmark.md     # 行业参考指标
```

## 周报内容框架（5 Parts）

| Part | 内容 | 数据来源 |
|------|------|---------|
| 1 | 账号数据总览 | 小红书后台「账号概览」截图 |
| 2 | 内容表现分析 | 小红书后台「笔记数据」截图 |
| 3 | 用户画像变化 | 小红书后台「粉丝数据」截图 |
| 4 | 竞品对标 | 公开渠道采集（第三方数据平台） |
| 5 | 下周选题建议 | 基于 Parts 1-4 的交叉分析 |

## 指标定义（美妆行业）

| 指标 | 计算方式 | 健康基准 |
|------|---------|---------|
| 互动率 | (点赞+收藏+评论)/观看量 | ≥ 3% 🟢, 1-3% 🟡, <1% 🔴 |
| 收藏率 | 收藏/观看量 | ≥ 1% 🟢（美妆收藏意愿强） |
| 涨粉效率 | 新增粉丝/观看量 | ≥ 0.5% 🟢 |
| 爆文率 | 互动率≥5% 的笔记占比 | ≥ 20% 🟢 |
| 笔记更新频率 | 周发布笔记数 | 3-5篇/周 🟢 |

## 竞品池（默认）

基于 Beauty Signal 定位（美妆），默认选取以下竞品：
- 同类新锐美妆品牌账号
- 美妆垂类达人账号（相似粉丝量级）
- 可替换：根据璐婷反馈调整竞品名单

## 交付标准

1. **时效**：每周二/三交付上周周报
2. **格式**：PDF（适配打印和移动端查看）
3. **可视化**：Chart.js 图表 + 数据卡片 + 表格
4. **洞察**：每份周报至少包含 3 个数据洞察（含 1 个反直觉洞察）
5. **行动**：Part 5 的选题建议需具体到标题方向 + 发布时间 + 优先级
